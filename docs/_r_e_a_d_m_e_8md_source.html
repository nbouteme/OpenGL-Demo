8 ## Dépendances
9

11 - GLM

15 
16 ### Matériel
17 
18 Le GPU doit supporter le standard OpenGL 3.3 (Disponible sur les GPU intégrés Intel avec les derniers pilotes, depuis au moins Sandy Bridge, le pilote libre nouveau pour GPU nVidia aussi <sub><sup>même si les performances sont terribles</sub></sup>, et les GPU intégrés sur les AMD-A4 bas de gamme)
19 
20 Bref, vous n'avez aucune excuses.
21 
22 Pour compiler:
23 
24 Faire une fois:
25 `git clone https://github.com/nbouteme/OpenGL-Demo`
26 
27 `cd OpenGL-Demo`
28 `mkdir build`
29 `cd build`
30 `cmake ..`
31 
32 Puis à chaque modification du code:
33 `` make -j`nproc` ``
34 dans le répertoire "build"

59 ## Support
60 
61 En théorie, le projet est multi-plateforme, en pratique, il a été compilé seulement sous Ubuntu et Arch Linux.
62 Vous devriez arriver à compiler sous Windows avec un environnement type-Unix (a.k.a. MSYS2 ou Cygwin)
63 
64 ### Installer les dépendances sous Ubuntu ~~(a.k.a. pls halp imma noob)~~
65 
66 Travis exécutant un script sur une machine virtuelle Ubuntu, vous pouvez vous référer au fichier .travis.yml pour une liste des commandes à taper dans un terminal.
67 Cela devrait suffire pour les dépendances.
68 
69 Les utilisateurs de distributions plus avant-gardistes (lire: non basée sur Debian) peuvent se débrouiller :^)
